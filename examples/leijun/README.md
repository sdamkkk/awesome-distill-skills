## 概述

这是雷军（小米创始人）思维蒸馏的完整成果，由「收集→提炼→生成→验证」四步法从 ~950KB 的公开演讲、深度专访和商业著作中蒸馏而来。

## 📦 文件说明

| 文件 | 用途 | 规模 |
|------|------|------|
| [`leijun-skill.md`](leijun-skill.md) | **v1.1 System Prompt** — 完整的 AI Skill 文件，可直接用于 LLM | 458 行, ~30KB |
| [`knowledge_card.md`](knowledge_card.md) | **知识卡** — 57 个结构化提取条目（决策框架/思维模型/表达风格/核心信念/追问应对） | 319 行 |
| [`validation_report.md`](validation_report.md) | **验证报告** — 4 问验证法测试结果，含评分和改进建议 | 356 行 |

## 🚀 如何使用

1. 复制 `leijun-skill.md` 的全部内容
2. 粘贴到任何 LLM 的 System Prompt 中（Claude / GPT / 等）
3. 开始对话 — 你可以问：
   - "雷军会怎么看这个产品定价策略？"
   - "我要做一个新品类，雷军会怎么建议？"
   - "如果小米来做这件事，雷军会怎么决策？"

也可以用作 Claude Code 的 `.skill` 文件或 OpenClaw 的 Skill。
